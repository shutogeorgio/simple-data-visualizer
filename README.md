## Data Visualizer With Pandas Data Reader

### Setup Environment
At working directory, run
```shell
$ ./init.sh
```
(configure Env Settings  & Install and Run Jupyter Notebook)


### Rerun Jupyter
```shell
$ ./start.sh
```
(Run Jupyter Notebook)
